This is a Sublime Text Main.sublime-menu customization. It adds a small toolbar-like set of quick actions to Sublime’s top menu bar by inserting menu entries after the existing menu section. Personally, I use Sublime Text as Notepad replacement and don't have extensive needs for it. This is just to save a few keystrokes and is a work in progress.
